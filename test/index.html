Технические составляющие документа и правильная настройка каждого параметра, условно называется фактором ранжирования
⚙ Технические
факторы ранжирования
Поведение пользователя
1. Устранение зеркал сайта.
2. Составление/коррекция файла robots.txt.
3. Внедрение/коррекция «псевдостатической» (человекопонятные URL, ЧПУ, релевантные запросу) адресации страниц.
4. Устранение дублей страниц.
5. Настройка кода ответа сервера для несуществующих страниц (настройка кода ответа 404).
6. Проверка корректности работы сайта при отключении JavaScript.
7. Простановка тега meta="robots".
8. Простановка/корректировка размещения тега noindex.
9. Простановка/корректировка размещения атрибута nofollow.
10. Простановка/корректировка размещения канонических ссылок (атрибута "canonical" тега link).
11. Составление/коррекция XML-карты сайта.
12. Составление и коррекция HTML-карты сайта.
13. Устранение избыточных 30Х-редиректов.
14. Устранение битых ссылок.
15. Оптимизация страниц пагинации.
16. Алгоритмы генерации метатегов.
17. Микроразметка (чаще всего schema.org).
18. Корректировка вывода текстов на второй и последующих страницах пагинации.
19. Перевод сайта на протокол HTTPS.
20. Наличие favicon.
Оптимизация сайта под мобильные устройства
21. Создание турбо-страниц.
22. Создание AMP-страниц.
23. Создание или оптимизация мобильной версии сайта.
24. Проверка скорости загрузки страниц сайта и подготовка рекомендаций по ее увеличению.
25. Внедрение/коррекция микроразметки на сайте.
Оптимизация кода страниц
26. Устранение/добавление избыточных/отсутствующих тегов.
27. Чистота кода (отсутствие больших блоков CSS и JS).
28. Корректность использования заголовков H1-H6.
29. Семантическая верстка: валидность HTML-разметки (проверка кода на ошибки).
Внедрение перелинковки
30. Внедрение/корректировка элемента «путь к странице».
31. Внедрение/корректировка сквозного меню.
32. Внедрение блока «Вас заинтересует».
33. Анализ работы хостинга (зависимости времени загрузки от числа одновременных подключений, анализ uptime хостинга).
34. Настройка кода 304.
35. Оптимизация размера картинок и видео.
36. Оптимизация функционала поиска по сайту.
37. Оптимизация иных типов документов (doc/xls/pdf).
38. Место нахождения сервера относительно целевой аудитории.
39. Соседи по IP.
